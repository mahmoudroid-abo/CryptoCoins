# Crypto Currency App

Simple Android App Using coinpaprika API to fitch and Display Crypto Currencies and it's detials,
this project follows Clean Architecture and MVVM Architectural Pattern.


## Features and Libraries:
 
   - Kotlin.<br/>
   - Jetpack Cpompose.<br/>
   - Kotlin Coroutines.<br/>
   - Clean Architecture.<br/>
   - MVVM architectural pattern.<br/>
   - View Model .<br/>
   - Retrofit2 .<br/>
   - Dependency Injection With Dageer Hilt.<br/>
